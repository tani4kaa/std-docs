# Infrastructure Specification Document (ISD)

## 1. Середовище розгортання
Система базується на хмарній інфраструктурі **AWS** та використовує технологію **Docker** для ізоляції мікросервісів. Це забезпечує швидке розгортання та незалежність компонентів системи.

## 2. Компоненти інфраструктури
* **Nginx (Reverse Proxy):** виконує роль вхідної точки, забезпечує SSL-термінацію та розподіл статичного контенту.
* **Load Balancer:** автоматично розподіляє трафік між активними контейнерами для підтримки високої доступності.
* **Amazon S3:** використовується як хмарне сховище для надійного зберігання медіафайлів (зображень товарів).
* **PostgreSQL Service:** керована база даних для збереження інформації про замовлення та користувачів.

## 3. Принципи масштабування та безпеки
Масштабування реалізовано за принципом **Horizontal Scaling** (горизонтальне розширення), що дозволяє автоматично запускати нові екземпляри сервісу при зростанні навантаження.
Безпека забезпечується шляхом ізоляції бази даних у приватній мережі (Private Subnet) та використання протоколу HTTPS.
